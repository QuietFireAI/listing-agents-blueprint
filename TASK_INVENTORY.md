# TASK_INVENTORY - every task, decision, and action in scope (non-social)

The design rule this file serves: not preparing the agent is failure in
design. Every row maps to an owning agent and its coverage (playbook or
tuple layer). A task with no row is an uncovered crossing: the ambiguity
protocol handles it live, and the after-action proposes its tuple.
Social/media tasks are OUT OF SCOPE here by owner direction (later phase).
